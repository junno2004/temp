# 로또 번호 생성기

간단한 로또 번호 생성 웹 페이지입니다. 버튼을 누르면 1~45 사이의 숫자 6개가 생성됩니다.

## 사용 방법
- `index.html`을 더블클릭해 브라우저로 열기
- `번호 생성` 버튼을 눌러 번호 생성
- `다크모드/화이트모드` 버튼으로 테마 전환

## GitHub Pages로 배포하기
1. GitHub 레포지토리에서 **Settings → Pages**로 이동
2. **Build and deployment**에서 Branch를 `main` / `root`로 선택
3. 저장 후 몇 분 뒤 아래 주소로 접속

```
https://junno2004.github.io/temp/
```
